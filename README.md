# c64-sprites
Kickass scrolling text and sine sprites for the Commodore 64

## Pre-requisite
- Visual Studio Code
- KickAss (C64) plugin 1.5.0 by Captain JiNX
- VICE C64 emulator

## How to assemble
Open `teedjay.asm` in VS Code and selcet `KickAss (C64): Build and Run` from the `Command Palette`.
